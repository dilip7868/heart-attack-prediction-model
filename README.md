# ❤️ Heart Attack Prediction using Machine Learning

## Project Overview
This project predicts the likelihood of a heart attack using Machine Learning based on patient health data.

The model analyzes medical attributes such as age, sex, chest pain type, ECG results, exercise-induced angina, and other clinical features to estimate heart attack risk.

## Objective
The objective of this project is to build a machine learning model that can assist in predicting heart attack possibility using healthcare data.

## Dataset Features
The dataset includes features like:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Resting ECG
- Maximum Heart Rate
- Exercise Angina
- ST Slope

## Technologies Used
- Python
- Pandas
- Scikit-learn
- LabelEncoder
- StandardScaler
- Jupyter Notebook

## Machine Learning Workflow
### 1. Data Loading
Loaded the dataset using Pandas.

### 2. Data Preprocessing
- Encoded categorical values using Label Encoding
- Scaled numerical values using StandardScaler

### 3. Train-Test Split
Split dataset into training and testing sets.

### 4. Model Building
Used Logistic Regression classifier.

### 5. Prediction & Evaluation
Evaluated the model using:
- Accuracy Score
- Classification Report

## Model Performance
### Accuracy Score
**84.24%**

## Conclusion
This model successfully predicts heart attack risk with good accuracy using Logistic Regression.
